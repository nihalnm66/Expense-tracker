# Expense Tracker

A simple Expense Tracker app where users can add, view, and manage their daily expenses.  
Built using **Node.js**, **Express**, **MongoDB**, and **HTML/CSS/JavaScript**.

## Features
- Add and delete expenses
- Categorize expenses (Food, Travel, Bills, etc.)
- User login and signup with JWT authentication
- MongoDB database using Mongoose
- Simple frontend UI (HTML, CSS, JavaScript)

## Tech Stack
- Node.js, Express.js
- MongoDB, Mongoose
- HTML, CSS, JavaScript (vanilla)
